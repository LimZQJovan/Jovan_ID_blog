Week 2 Progress

This week, I accomplished the following tasks:

  1. Completed CA1 semantic exercise
    • Overall, the goal is to structure the HTML content in a logical and organized manner, making it easier to maintain and update.
    • By using the header, footer, section, and nav elements, the content is grouped into meaningful sections, making it easier to understand and work with.
  2. Completed CA2 semantic exercise vr
    • Overall, the goal is to make the HTML more meaningful and organized, using semantic markup to accurately describe the content and improve the accessibility and search engine optimization of the recipe blog.
  3. The existing external CSS file "style.css" is used to add style to the HTML content
    • To add the CSS file, the link method is used to link the HTML file to the CSS file.
    • This allows for the styling information to be stored separately from the HTML content, making it easier to maintain and update.
    • A few CSS properties are added to the HTML content to enhance its appearance.
    • These properties may include color, background-color, font-family, and text-align.
  4. The #heading ID is applied to an appropriate element.
    • By using the ID, the styles can be applied to a specific element on the page.
    • A class style is also applied to other appropriate elements, in order to change their appearance as well.
    • Classes allow for styles to be reused multiple times throughout the page, giving more control over the appearance of different elements.
    • Overall, the goal is to use the CSS styles to improve the appearance and make it more visually appealing to the user
  5. The process of adding Google fonts to the website by visiting fonts.google.com
    • On the website, the desired font(s) are selected by browsing through the available options and making a selection.
    • At least two Google fonts are chosen, one for headings and another for paragraphs, and the stylesheet is used to implement the selected fonts
    • The CSS stylesheet is updated to include the code necessary to apply the selected fonts to the appropriate elements, such as headings and paragraphs.
    • Overall, the goal is to add Google fonts to the recipe blog in order to enhance its appearance and create a more personalized typography style for the website.
